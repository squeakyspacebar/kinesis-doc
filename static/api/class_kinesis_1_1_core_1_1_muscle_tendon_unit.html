float Kinesis.Core.MuscleTendonUnit.SerialElasticForce ( float  seStrain)

Models the elastic spring force generated by the serial elastic element when its length extends beyond slack length.

To denormalize the result value, multiply by the max isometric contraction force.

float seStrain = this.seLength / this.seSlackLength - 1.0f; float normalizedSerialElasticForce = this.SerialElasticForce(seStrain);

Parameters

Returns
Restoring elastic force normalized by maximum isometric contraction force. Returns zero if the serial element's current length does not exceed slack length.
